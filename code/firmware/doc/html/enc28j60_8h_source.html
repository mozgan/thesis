38  * Control registers are generically grouped as ETH, MAC and MII registers.
39  * Register names starting with 'E' belong "ETH" group. Similarly, registers names starting with 'MA' belong to the "MAC" group and registers prefixed with 'MI' belong to the "MII" group.

41  * - Register address (bits 0-4)
42  * - Bank number (bits 5-6)
43  * - MAC/PHY indicator (bit 7)
