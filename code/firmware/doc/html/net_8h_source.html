30 /*
31  * Ethernet Frame Formats (v2)
32  * ---------------------------
33  *
34  * +----+----+------+------+-----+
35  * | DA | SA | Type | Data | FCS |
36  * +----+----+------+------+-----+
37  *
38  * Packet format:
39  * +-----------------------------------------------------+
40  * | Preamble | Dest. MAC | Src. MAC | Type | Data | FCS |
41  * +-----------------------------------------------------+
42  *
43  * Preamble (8 bytes)
44  * DA Destination MAC Address (6 bytes)
45  * SA Source MAC Address (6 bytes)
46  * Type Protocol Type (2 bytes)
47  * Data Protocol Data (46 - 1500 bytes)
48  * FCS Frame Checksum (4 bytes)
49  *
50  * Header:
51  * IP (the Internet protocol) : 0x0800
52  * ARP : 0x0806
53  *
54  * source:
55  * http://tools.ietf.org/id/draft-kaplan-isis-ext-eth-02.txt
56  * http://www.infocellar.com/networks/ethernet/frame.htm
57  * http://wiki.wireshark.org/Ethernet
58  */

135  * Internet Control Message Protocol (ICMP)
136  * ----------------------------------------
137  *
138  * Message Formats:
139  * Version: 4
140  * IHL: Internet header length in 32-bit words.
141  * Type of Service: 0
142  * Total Length: Length of internet header and data in octets.
143  * Identification, Flags, Fragment Offset: see http://tools.ietf.org/html/rfc791
144  * Time to Live (TTL): Time to live in seconds.
145  * Protocol: ICMP = 1
146  * Header Checksum: The 16 bit one's complement of the one's complement
147  * sum of all 16 bit words in the header.
148  * Source Address: The address of the gateway or host that composes the
149  * ICMP message.
150  * Destination Address: The address of the gateway or host to which the
151  * message should be sent.
